BSML Objects

Perl object layer for BSML (Bioinformatic Sequence Markup Language)

Description

BSML is an XML format for encoding a wide variety of information attached to biological sequences. This project provides a set modules for representing BSML adherent to v3.1 of the BSML DTD in Perl. The Perl modules include the ability to parse and generate BSML documents.

Background

BSML is an open XML data standard developed by Labbook. The Perl BSML Objects were developed by the Bioinformatics Department at The Institute for Genomic Research (TIGR)

Requirements

  • Perl 5.6.0 or above
  • XML::Twig

BSML:: Perl Module Install

    The BSML:: Perl modules are installed using the standard Perl module install syntax. In addition, you must specify a SCHEMA_DOCS_DIR. This directory will contain the v3.1 DTD and XML schemas that are distributed with the package. 
      cvs co bsml_install
  cd bsml_install
  perl Makefile.PL PREFIX=[install_dir] SCHEMA_DOCS_DIR
  make 
  make install
