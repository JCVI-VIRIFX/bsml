Sequences

Naming convention

The identifier for the sequence is generated using the field assembly.asmbl_id.

//Sequence/@id = database_name+assembly.asmbl_id+"_assembly"

Example:
gba_1599_assembly

Note: All subfeatures of the sequence will use a similar naming convention to generate //Feature/@id. The naming convention for //Sequence/@id and //Feature/@id will likely change. Software should not make any assumptions about the naming conventions.

Assembly sequence

//Sequence/@id = database_name + "_" +assembly.asmbl_id +"_assembly" //Sequence/@class = `assembly' //Sequence/@molecule = `dna' //Sequence/@length = datalength(assembly.sequence) //Sequence/@topology = asmbl_data.topology where assembly.asmbl_id = stan.asmbl_id and stan.asmbl_data_id = asmbl_data.id //Sequence/Attribute[@name='molecule_name'] = asmbl_data.name where assembly.asmbl_id = stan.asmbl_id and stan.asmbl_data_id = asmbl_data.id //Sequence/Attribute[@name='molecule_type'] = asmbl_data.type where assembly.asmbl_id = stan.asmbl_id and stan.asmbl_data_id = asmbl_data.id //Sequence/Attribute-list/Attribute[@name='SO']/@value = `supercontig' where assembly.asmbl_id = stan.asmbl_id and stan.asmbl_data_id = asmbl_data.id and asmbl_data.name like `%pseudomolecule%' //Sequence/Attribute-list/Attribute[@name='SO']/@value = `plasmid' where assembly.asmbl_id = stan.asmbl_id and stan.asmbl_data_id = asmbl_data.id and asmbl_data.name like `%plasmid%' //Sequence/Attribute-list/Attribute[@name='SO']/@value = `chromosome' where assembly.asmbl_id = stan.asmbl_id and stan.asmbl_data_id = asmbl_data.id and asmbl_data.name like `%chromosome%' //Sequence/Seq-data->text() = assembly.sequence

The following sections describe elements which are nested below the assembly sequence described above.

Gene encoding

The ORF(or NTORF) from the annotation database is used to fill the required features for the standard Chado gene encoding. The Chado gene encoding requires 5 <Feature> elements: gene, transcript, CDS, exon, and protein. Note: <Interval-loc> and <Cross-reference> elements for each of the Gene, Transcript, CDS, and Exon <Feature> elements are identical.

Gene Feature
