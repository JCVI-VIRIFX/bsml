TIGR-BSML to Bioperl Adapter



Description
The TIGR-BSML to Bioperl adapter is a set of Bioperl modules that use the BSML object layer to allow Bioperl scripts to read and write BSML files that adhere to the BSML encoding conventions used at TIGR.
