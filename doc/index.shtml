BSML



Description
BSML is an XML-based format for storing bioinformatic data. The BSML 3.1 specification includes encodings for Information (e.g., genomes, biological sequences, search hits, and alignments), Research (e.g., the queries, analyses, and experiments used to derive Information), and Visualization (e.g., content that specifies how the Information and Research data should be rendered in a graphical display.) The BSML object layer being used at TIGR (see CVS links below) deals only with subsets of the first two encodings, Information and Research. XML Schema specifications (see below) are used to ensure that the BSML documents used in our software systems adhere to the specific subset of BSML with which we are working.

Etymology/History
BSML is an acronym that stands for "Bioinformatic Sequence Markup Language". See the BSML.org and LabBook sites for more information on the history of BSML. Note that LabBook distributes tools for displaying and/or browsing BSML documents in a graphical context, hence the inclusion of BSML's Visualization encoding.
