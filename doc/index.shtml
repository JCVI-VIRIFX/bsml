BSML



Description
BSML is an XML-based format for storing bioinformatic data. The BSML 3.1 specification includes encodings for "Information" (e.g., genomes, biological sequences, search hits, and alignments), "Research" (e.g., the queries, analyses, and experiments used to derive Information), and "Visualization" (e.g., content that specifies how the Information and Research data should be rendered in a graphical display.) The BSML object layer being used at TIGR (see CVS links below) deals only with subsets of the first two encodings, Information and Research.

Etymology/History
BSML is an acronym that stands for "Bioinformatic Sequence Markup Language". See the BSML.org and LabBook sites for more information on the history of BSML.
